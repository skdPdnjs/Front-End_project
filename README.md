# Front-End 개발실습 과제
 - 학점관리 사이트 / 사용 언어: JavaScript, HTML, CSS

 - 사이트 관련 추가설명
   1. 추가버튼 클릭시 입력 칸 생성
   2. 1학점(기본 세팅) -> 무조건 선택은 P, NP로 구성(저장 누르면 텍스트로 변환)
   3. 2학점, 3학점 선택 시 점수 입력 생성
   4. 저장 버튼 누르면 계산
   5. F는 학점 계산에서 제외, 만약 재수강을 통해 같은 과목이 2개 있어도 F학점 아닌 것만 합산 후 계산 

- 이외 첨부문서에 있는 조건 모두 충족
